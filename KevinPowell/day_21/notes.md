# Day 21: Recap
What we've learned:
- Website are responsive before we write any css.
- When our layouts run into issues, we're at fault.
- Usually (though not always), a desktop-first approach is the culprit.
- Writing mobile-first CSS tends to be the easier way to approach it as well, even if you only have a desktop layout to base things off of.
- With just doing the typography and spacing, the mobile layout is already mostly done usually.

Where to go from here:
- We looked at the basics of flexbox, but there is a lot more to it.
- And you might have heard about grid as well.
- Don't rush things.
- Mastering anything isn't about learning as much as you possibly can.
